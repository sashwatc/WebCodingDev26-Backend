# NLC Demo Script

Target time: 2 to 4 minutes.

1. Open `/EventHub?id=hub_basketball_game` and show the PVHS vs. Bettendorf Basketball Game recovery hub.
2. Open Gym Bleachers beacon and submit a Lost Report for an AirPods-style case.
3. Open User Dashboard and show the Recovery Case message: staff are checking likely locations.
4. Switch to Admin Dashboard, Recovery Center tab, and show likely zones plus mission controls.
5. Open the AirPods-style Found Item and show public custody timeline plus Integrity verified.
6. Submit a claim and point out Sealed Ownership Verification fields.
7. In Admin Dashboard, show Proof Vault and Claim Evidence Review as admin-only data.
8. Create or open the seeded active Return Pass, then open `/PickupPass?id=pass_lanyard_active`.
9. Go to `/PickupStation`, verify manual code `314159`, and redeem the pass.
10. Show returned status and custody ledger verification.
11. Close with Loss Sentinel alert and Partner Relay simulation in Admin Dashboard.

Working functionality: reports, claims, recovery cases, missions, proof evidence, custody ledger verification, display feed, asset lookup, return pass verification/redemption, sentinel alerts, and relay summaries.

Demo/integration-ready simulation: event source, beacon placement, display hardware, asset registry, and partner relay networking.
